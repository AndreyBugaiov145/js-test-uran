1. Задание 1

Реализовать класс TagList.

Конструктор принимает первым аргументом узел в который будет помещен html виджета, и вторым опциональным параметром список предопределенных тегов. Виджет имеет 2 режима отображения: показ и редактирование. В режиме редактирования можно добавлять и удалять теги. Добавлять уже существующие теги нельзя. Добавление происходит по enter и по клику на кнопку "добавить". Должен быть реализован метод getTagList, возвращающий список тегов (массив строк).

написать стили самостоятельно

библиотеки использовать нельзя

можно использовать bootstrap (только css), если есть желание
